## Summary

The scan-for-secrets 0.2 CLI tool now streams results in real-time, supports multiple directory scans, and introduces new API functions and options for enhanced usability.

## Full Article

CLI tool now streams results as they are found rather than waiting until the end, which is better for large directories. -d/--directory option can now be used multiple times to scan multiple directories. New -f/--file option for specifying one or more individual files to scan. New scan_directory_iter(), scan_file() and scan_file_iter() Python API functions. New -v/--verbose option which shows each directory that is being scanned.
