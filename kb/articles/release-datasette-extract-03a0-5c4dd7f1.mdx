## Summary

The release of datasette-extract 0.3a0 introduces the ability to import unstructured data into structured tables, utilizing datasette-llm for model management.

## Full Article

Release datasette-extract 0.3a0 — Import unstructured data (text and images) into structured tables This plugin now uses datasette-llm to configure and manage models. This means it's possible to specify which models should be made available for enrichments, using the new enrichments purpose.
