## Summary

The release of datasette-agent-charts 0.1a1 introduces enhanced color options for charts, interactive tooltips, and improved permission checks for SQL queries.

## Key Insights

> "More color! Bar and waffle charts without a color column are shaded by magnitude with a sequential color scheme."
>
> — Introduction of new color features in charts.

> "Now checks execute-sql permission before running the query to find the column names."
>
> — Improvement in security and functionality.

> "Charts now display interactive tooltips."
>
> — Enhancement of user interaction with charts.

## Full Article

More color! Bar and waffle charts without a color column are shaded by magnitude with a sequential color scheme; color columns holding text values use the observable10 categorical scheme. #2 Now checks execute-sql permission before running the query to find the column names. Charts now display interactive tooltips. Fixed a bug where waffleY charts were not described to the agent.
