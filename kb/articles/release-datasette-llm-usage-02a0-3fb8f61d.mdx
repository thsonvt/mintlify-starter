## Summary

The release of datasette-llm-usage 0.2a0 includes the removal of certain features and introduces new logging capabilities for prompts and responses.

## Full Article

Removed features relating to allowances and estimated pricing. These are now the domain of datasette-llm-accountant. Now depends on datasette-llm for model configuration. #3 Full prompts and responses and tool calls can now be logged to thellm_usage_prompt_log table in the internal database if you set the new datasette-llm-usage.log_prompts plugin configuration setting. Redesigned the /-/llm-usage-simple-prompt page, which now requires the llm-usage-simple-prompt permission.
