## Full Article

Release datasette-llm 0.1a7 — LLM integration plugin for other plugins to depend on Mechanism for configuring default options for specific models. Part of Datasette's evolving support mechanism for plugins that use LLMs. It's now possible to configure a model with default options, e.g. to say all enrichment operations should use a specific model with temperature set to 0.5.
