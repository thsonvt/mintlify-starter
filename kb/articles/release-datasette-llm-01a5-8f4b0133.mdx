## Summary

The release of datasette-llm 0.1a5 introduces a plugin hook that tracks prompts in tool call loops, enhancing LLM integration for plugins.

## Key Insights

> "The llm_prompt_context() plugin hook wrapper mechanism now tracks prompts executed within a chain."
>
> — Describing the new features of the datasette-llm 0.1a5 release.

> "It can be used to track tool call loops."
>
> — Highlighting the functionality of the llm_prompt_context() plugin hook.

## Full Article

Release datasette-llm 0.1a5 — LLM integration plugin for other plugins to depend on The llm_prompt_context() plugin hook wrapper mechanism now tracks prompts executed within a chain as well as one-off prompts, which means it can be used to track tool call loops. #5
