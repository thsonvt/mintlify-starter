## Summary

Romain Huet confirms that OpenAI has unified Codex and the main model, with GPT-5.5 enhancing coding capabilities further.

## Full Article

25th April 2026 Since GPT-5.4, we’ve unified Codex and the main model into a single system, so there’s no separate coding line anymore. GPT-5.5 takes this further, with strong gains in agentic coding, computer use, and any task on a computer. — Romain Huet, confirming OpenAI won't release a GPT-5.5-Codex model
